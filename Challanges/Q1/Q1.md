


## Coding Challenge: The Smart Cart Calculator

### **Problem Description**

You are building the backend for a smart shopping cart. The cart receives a list of items, calculates discounts based on certain conditions, filters out invalid items, and outputs a final receipt summary.

To solve this, you will need to process different data types, use loops to iterate through items, apply conditional logic for discounts, and wrap your logic inside functions.

---

### **Input Format**

You will be given a list of dictionaries (representing the items in the cart) and a user status string (`"Premium"` or `"Regular"`).

Each item dictionary contains:

* `"name"`: (String) The name of the item.
* `"price"`: (Float/Integer) The price of a single unit.
* `"quantity"`: (Integer) How many units were bought.
* `"category"`: (String) e.g., "Electronics", "Groceries", "Clothing".

---

### **Constraints**

* If an item's price or quantity is less than or equal to `0`, it is considered invalid and must be ignored.
* If the total cart value exceeds `$100`, a flat **10% discount** is applied to the final bill.
* If the user is `"Premium"`, they get an **additional 5% discount** on top of any other discounts.

---

### **Output Format**

Your function must return a **Tuple** containing three elements:

1. A **List** of strings containing only the names of the *valid* items purchased.
2. A **Set** of unique categories present in the valid items.
3. The **Final Price** (Float) rounded to 2 decimal places.

---

### **Sample Input**

```python
cart_items = [
    {"name": "Laptop", "price": 80.00, "quantity": 1, "category": "Electronics"},
    {"name": "Apple", "price": 1.50, "quantity": 4, "category": "Groceries"},
    {"name": "Broken Pen", "price": -5.00, "quantity": 1, "category": "Stationery"}, # Invalid price
    {"name": "Shirt", "price": 20.00, "quantity": 2, "category": "Clothing"},
    {"name": "Banana", "price": 0.50, "quantity": 2, "category": "Groceries"}
]
user_status = "Premium"

```

### **Sample Output**

```python
(['Laptop', 'Apple', 'Shirt', 'Banana'], {'Electronics', 'Groceries', 'Clothing'}, 107.73)

```

### **Explanation**

1. **Filtering & Loops:** The loop skips "Broken Pen" because its price is negative.
2. **Data Types & Math:** * Laptop: $80 \times 1 = 80$
* Apple: $1.5 \times 4 = 6$
* Shirt: $20 \times 2 = 40$
* Banana: $0.5 \times 2 = 1$
* *Subtotal* = $80 + 6 + 40 + 1 = 127.0$ (Float)


3. **Conditions:** * Since $127 > 100$, a 10% discount is applied ($127 \times 0.90 = 114.3$).
* Since the user is `"Premium"`, an additional 5% discount is applied ($114.3 \times 0.95 = 108.585$).


4. **Final Collections:** The unique categories are gathered into a `set` (removing duplicate "Groceries").

---

## Starter Code Template

Your friend can copy and paste this into their IDE to solve it:

```python
def calculate_receipt(items: list, status: str) -> tuple:
    # 1. Initialize your data types here
    valid_item_names = []  # List
    unique_categories = set()  # Set
    subtotal = 0.0  # Float

    # 2. Loop through the items
    for item in items:
        # Extract variables from the Dictionary
        name = item["name"]
        price = item["price"]
        quantity = item["quantity"]
        category = item["category"]

        # 3. Apply Conditional logic to filter out invalid data
        if price <= 0 or quantity <= 0:
            continue  # Skip this item

        # 4. Perform calculations and update data structures
        valid_item_names.append(name)
        unique_categories.add(category)
        subtotal += price * quantity

    # 5. Apply conditional discounts
    if subtotal > 100:
        subtotal *= 0.90  # 10% off

    if status == "Premium":
        subtotal *= 0.95  # 5% off

    # 6. Return the required Tuple, rounding the final price
    return (valid_item_names, unique_categories, round(subtotal, 2))
```
